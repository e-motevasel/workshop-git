سوال 4: تفاوت Local repository و Remote repository
Local repository  روی سیستم هر فرد هست و برای زدن کد ها استفاده میشه و ارتباطی با سرور نداره اما remote repository  روی سروره و بر خلاف لوکال، برای کار گروهیه

سوال 5: دستور git status
این دستور نشون میده که الان پروژه توو چه وضعیتیه. مثلا نشون میده چه فایلایی تغییر کردن یا چه فایلایی نیاز به کامیت دارن یا چه فایلایی استیج شدن ولی هنوز کامیت نشدن و ...

سوال 6: کار نکردن به صورت مستقیم روی main
علت این کار اینه که برنچ مین همیشه باید سالم و آماده ارائه باشه. اگه مستقیم روی برنچ کار کنیم ممکنه کد از کار بیفته. همینطور برگشت به نسخه های قبلی سخت تر میشه و برنچ کثیفتر میشه. همینطور این کار امکان بازبینی رو ازمون میگیره.
